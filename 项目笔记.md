###解决移动端响应延迟问题
```js
<script src="js/fastclick.js"></script>
  <script>
        if ('addEventListener' in document){
            document.addEventListener('DOMContentLoaded',function (){
                FastClick.attach(document.body)
            })
        }
  </script>
```
###使用fastClick时。文件中mode :'history'会报错。改为‘hash’不会报错
